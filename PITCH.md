## Slide 2 — The Problem We Solved

**We were flying blind on where our hours go.**

- No unified way to track billable vs. non-billable work

## Slide 3 — What We Built

**A custom time tracking platform built for Duran File Pino — not a generic SaaS.**

- Real-time timer with live tracking (centisecond precision)
- Multi-workspace support for different teams/departments
- Billable rate configuration per project and member
- Preset timer configurations (10 per workspace) for repeat tasks

> Built in-house. Tailored to how we actually work.

## Slide 8 — Tech Foundation (For Technical Stakeholders)

- **Frontend:** React SSR (TanStack Start), Tailwind CSS, shadcn/ui
- **Backend:** PostgreSQL (Neon serverless), Drizzle ORM, Better Auth
- **Hosting:** Serverless — scales with the team, no maintenance overhead
- **Security:** Token-based workspace invites, role-based access, email authentication

> Built on production-grade, maintained open-source tooling — not a custom stack we'd have to support from scratch.
